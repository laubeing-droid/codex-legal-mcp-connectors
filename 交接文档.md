## 一、定位

独立的 MCP 连接器中心。职责：管理本机 MCP 配置 + 托管自建 MCP Server。被 [Claude-for-Legal-CN-to-Codex](https://github.com/laubeing-droid/Claude-for-Legal-CN-to-Codex) 委托调用。

## 二、文件结构


## 四、维护场景

**npm 包更新**：`npm-monitor.yml` 自动检查 `@pkulaw/mcp-cli` 新版并创建 Issue。

**新增客户端**：修改 detect.ps1/sh → 写入逻辑 → 文档。

**跨平台**：修改逻辑时 PowerShell + Bash 同步更新。
